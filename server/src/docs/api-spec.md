# API Spec

Base URL: `/api`

- GET `/scenes`
- GET `/scenes/:sceneKey`
- GET `/auth/me`
- POST `/upload`
- POST `/images/generate`
- GET `/tasks/:taskId`
- GET `/images/history?page=1&pageSize=10`
- GET `/images/:imageId/detail`
- POST `/orders`
- GET `/orders/:orderId`
- POST `/orders/:orderId/mock-pay`
- GET `/download/:resultId/preview`
- GET `/download/:resultId/hd`
- GET `/download/:resultId/print`
- GET `/admin/stats`
